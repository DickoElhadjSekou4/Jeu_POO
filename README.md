# Jeu_POO

Projet réalisé en Ruby dans le cadre de l'apprentissage de la Programmation Orientée Objet.

## Description

"ILS VEULENT TOUS MA POO" est un jeu en ligne de commande dans lequel le joueur incarne un personnage humain qui doit survivre face à plusieurs ennemis.

Le but du jeu est simple : être le dernier survivant.

Le joueur peut :

* Chercher une meilleure arme
* Chercher à se soigner
* Attaquer un ennemi visible

Les ennemis apparaissent progressivement et attaquent à chaque tour tant qu’ils sont en vie.

## Fonctionnalités implémentées

Version 1 :

* Création de la classe Player
* Système d’attaque entre joueurs
* Gestion des points de vie

Version 2 :

* Ajout de la classe HumanPlayer
* Menu d’actions dynamique
* Gestion d’ennemis multiples
* Condition de victoire ou de défaite

Version 3 :

* Création de la classe Game
* Apparition progressive des ennemis
* Gestion du nombre total de joueurs restants
* Structure complète orientée objet
* Boucle de jeu contrôlée par la classe Game

Depuis le terminal, placer-vous dans le dossier du projet puis exécuter :
